# Instapy-Tools

Want to build on top of Instapy? Then this is the toolset for you!

Requirements:
  - node > 10

## Usage

#### Cloning InstaPy Project

```
npm run instapy:clone
```

#### Updating InstaPy

```
npm run instapy:update
```

#### Generate api.json

```
npm run gen-api
```

#### Develop api.json

```
npm run gen-api:dev
```
